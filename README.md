# Offroad
![Build](https://github.com/radevgit/offroad/actions/workflows/rust.yml/badge.svg)

## Adding the library to your project

To use the Offroad library in your project, add the following to your `Cargo.toml`:

```toml
[dependencies]
offroad = "0.0.5"
```
